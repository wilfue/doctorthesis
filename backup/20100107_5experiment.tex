\chapter{セレクトブクマの評価}
	\section*{概要}
本章では，セレクトブクマに対して行った実験とその評価結果について示す． さらに，評価結果を分析し，分析結果も示す．
	\newpage
	\section{実験概要}
セレクトブクマの有用性を評価するため，以下3種類の手法において比較実験を行った．
\begin{itemize}
\item Google検索
\item 取得したはてなブックマークのデータの中で，タグで検索した場合のブックマーク数が多いものから順にランキングしたもの（以下，ブックマーク数順）
\item セレクトブクマ（α=1）
\end{itemize}

	\section{実験目的}
本実験の目的としては，本研究の目的である，Web上から以下の情報を手軽に取得できるかどうかについて評価することである．
\begin{itemize}
\item 体系だった知識を得ること
\item 有益なWebコンテンツを発見すること
\end{itemize}
そのため，以下のようなことを目的とした実験をおこなった．
\begin{itemize}
\item セレクトブクマと既存の検索エンジンにおいて，どちらがより今後も利用するような体系だった知識を得られるWebページを手軽に取得できるか評価する
\item セレクトブクマで利用している理論である，時間情報による重み付けによって，一時的に必要とされる情報をフィルタリングできていることを実証する
\end{itemize}

	\section{実験方法}
以下，3種類の手法で検索を行い，検索結果上位30件を取得した．取得した合計90件のWebサイトの中で，被験者に今後も利用したいと思うWebサイトを1位〜10位まで選んでもらった．
\begin{itemize}
\item Google検索
\item ブックマーク数順
\item セレクトブクマ（α=1）
\end{itemize}

被験者が選んだWebサイトを正解集合として，適合率，再現率，F値を求める．また，同様に時間係数αの値を変更していって，αの値による適合率，再現率を求め，最適なαの値を求める．

実験条件は以下とする．
\begin{itemize}
\item 被験者数：30人（1単語につき10人×3単語）
\item 検索単語（タグ）：「java」，「健康」，「映画」
\item 「java」：男性6人，女性4人
\item 「健康」：男性5人，女性5人
\item 「映画」：男性3人，女性7人
\end{itemize}
	\section{実験結果}
